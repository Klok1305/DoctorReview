# Краткая инструкция

## Первый запуск

1. Дважды нажмите `ОценкаВрачей-1.0.3-x64.exe` и завершите установку.
2. Запустите ярлык «Оценка врачей» на рабочем столе.
3. Выберите рабочую папку. Используйте локальную папку вне OneDrive. Если не уверены, оставьте предложенную папку `C:\Users\<имя пользователя>\Оценка врачей`.

Приложение само создаст внутри неё папки «Входящие», «База», «Результаты», «Резервные копии» и «Журналы».

## Обычная работа

1. Скопируйте выгрузки 1С (`.xls`, `.xlsx` или `.zip`) в папку «Входящие».
2. Откройте приложение и нажмите «Обработать новые выгрузки».
3. Проверьте таблицу «Данные в базе» и журнал загрузок.
4. На вкладке «Отчёт» выберите месяц и нажмите «Выгрузить все PDF».
5. Готовые файлы находятся в `Результаты\ГГГГ-ММ`.

В папке месяца создаются отдельные файлы для каждого врача и каждого отделения. Если выгрузить месяц повторно, прежние PDF переместятся в `Предыдущие версии`, поэтому в основной папке останется только актуальный комплект.

## Если нужно поменять папки

На вкладке «Данные» в блоке «Рабочее место» можно отдельно изменить:

- рабочую папку целиком;
- папку входящих файлов;
- папку результатов;
- папку резервных копий.

При переносе рабочей папки текущая база копируется в новое место. Исходная база остаётся на старом месте как дополнительная страховка.

## Резервные копии

- «Создать копию» — сохраняет проверенную копию в настроенную папку резервов.
- «Переносимая копия» — создаёт один файл `.ovbackup`, который можно перенести на другой компьютер.
- «Восстановить» — заменяет рабочую базу выбранной копией. Перед заменой приложение автоматически сохраняет текущее состояние.

## Обновление

Получив новый установщик, откройте приложение, нажмите «Установить из файла» и выберите новый `.exe`. Приложение сначала создаст резервную копию базы, затем запустит обновление. Рабочая папка и все данные сохранятся.

Если Windows показывает SmartScreen для неподписанной сборки, нажмите «Подробнее» → «Выполнить в любом случае». Для публичного распространения рекомендуется подписать установщик сертификатом.
